CustomGif Component Documentation

The CustomGif component allows you to display GIFs with customizable styles. Below are examples demonstrating its usage and the inputs it accepts.

Accepted Inputs

  • src (required): The source URL of the GIF.
    Example: assets/gifs/ai-blue-brain.gif
  • alt (optional): Alternative text for the GIF.
    Default: 'GIF Image'
  • customStyles (optional): A key-value pair object for custom styles.
    Example: {{ "{" }} border: '2px solid #4caf50', borderRadius: '10px', boxShadow: '0 4px 8px rgba(0, 0, 0, 0.2)', margin: '20px auto', maxWidth: '300px' {{ '}' }}

Custom Styles Applied:

  • border: '2px solid #4caf50': Adds a green border with a thickness of 2 pixels, making the GIF visually stand out.
  • borderRadius: '10px': Rounds the corners of the GIF, giving it a softer, modern appearance.
  • boxShadow: '0 4px 8px rgba(0, 0, 0, 0.2)': Applies a subtle shadow effect to the GIF, creating a sense of depth and elevating it above the background.
  • margin: '20px auto': Centers the GIF horizontally and adds a 20-pixel space around it for better spacing in layouts.
  • maxWidth: '300px': Limits the maximum width of the GIF to 300 pixels, ensuring it fits within smaller layouts while maintaining responsiveness.

Effect: The GIF is styled to stand out visually while maintaining responsiveness and alignment in various layouts.
