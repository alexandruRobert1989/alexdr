CustomVideo Component Documentation

The CustomVideo component allows you to embed videos with customizable playback options and styles. Below are examples demonstrating its usage and the inputs it accepts.

Accepted Inputs

  • src (required): The source path to the video file.
    Example: assets/videos/ai-blue-brain.mov
  • poster (optional): A poster image displayed before the video starts playing.
    Example: assets/images/video-poster.jpg
  • controls (optional): Whether to show playback controls.
    Default: true
  • autoplay (optional): Whether the video should start playing automatically.
    Default: false
  • loop (optional): Whether the video should loop after finishing.
    Default: false
  • muted (optional): Whether the video should start muted.
    Default: false
  • allowDownload (optional): Allow the "Download" option in controls.
    Default: true
  • customStyles (optional): Key-value pair object for custom styles.
    Example: {{ '{' }} border: '2px solid blue', maxWidth: '400px' {{ '}' }}

Custom Styles and Settings Used:

  • border: '2px solid blue': Adds a 2-pixel solid blue border around the video.
  • max-width: '400px': Restricts the maximum width of the video to 400 pixels, ensuring it scales proportionally.
  • autoplay: true: Automatically starts playing the video when the component is loaded, creating a dynamic and engaging experience.
  • loop: true: Ensures the video automatically restarts from the beginning once it reaches the end, providing continuous playback.
  • muted: true: Starts the video with no sound, making it ideal for scenarios where autoplay is enabled to avoid abrupt noise.
  • allowDownload: false: Disables the "Download" option in the video controls, restricting users from downloading the video directly from the player.
Effect:
  • The video is visually highlighted with a blue border, and its size is constrained to fit within 400px width for better layout control.
  • The video starts playing automatically upon component load.
  • The video loops continuously without requiring manual restart.
  • The video starts muted, preventing any initial sound playback.
  • The "Download" option is removed from the video controls, ensuring content protection.
