CustomLottie Component Documentation

The CustomLottie component allows you to integrate Lottie animations with customizable options and styles. Below are examples demonstrating its usage and the inputs it accepts.

Accepted Inputs

  • options (required): The configuration options for the Lottie animation.
    Example: {{ "{" }} path: 'assets/animations/rocket.json', renderer: 'svg', autoplay: true, loop: true {{ "}" }}
  • speed (optional): The playback speed of the animation.
    Default: 0.5
  • direction (optional): The playback direction of the animation.
    Default: 1 (forward); use -1 for reverse.
  • loop (optional): Whether the animation should loop.
    Default: true
  • customStyles (optional): A key-value pair object for custom styles applied to the animation container.
    Example: {{ "{" }} maxWidth: '300px', maxHeight: '300px', border: '1px solid black' {{ "}" }}
    Explanation:
    • maxWidth: '300px': Limits the maximum width of the animation container to 300 pixels.
    • maxHeight: '300px': Limits the maximum height of the animation container to 300 pixels.
    • border: '1px solid black': Adds a black border with a thickness of 1 pixel around the animation.

Custom Styles Applied:

  • maxWidth: '300px': Restricts the animation container's maximum width to 300 pixels.
  • maxHeight: '300px': Restricts the animation container's maximum height to 300 pixels.
  • border: '1px solid black': Adds a 1-pixel solid black border around the container.

Effect: The Lottie animation is styled to fit within a 300x300 pixel container, centered and visually framed by a black border for better emphasis.
