Directive Documentation

The AnimationDirective applies animation classes to elements. Below is the list of accepted inputs:

  • [appAnimation] (required): The animation type, based on the AnimationType enum.
    Example: AnimationType.Bounce

      [appAnimation]="AnimationType.Bounce"
